# compass-ble

Папка содержит контент проекта для портфолио.

- `compass-ble.md` — основной файл (YAML frontmatter + markdown тело).
- `cover.webp` — обложка карточки (не обязательно; при отсутствии показывается плейсхолдер).
- `architecture.webp` — схема архитектуры, на которую ссылается тело md.
- любые дополнительные картинки кладутся рядом и подключаются относительным путём `assets/projects/compass-ble/<file>`.

При добавлении новых картинок убедитесь, что папка указана в `pubspec.yaml` → `flutter.assets`.
